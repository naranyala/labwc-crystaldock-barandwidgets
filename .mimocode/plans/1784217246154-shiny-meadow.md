## Dock.c toggle bar implementation

Port the cairo-pango dock.zig toggle logic to C:

```c
// After drawing centered icons, add separator + toggles
int toggle_start = start_x + total_w + DOCK_PAD;
int divider_x = start_x + total_w + DOCK_PAD / 2;

// Vertical separator line
blend_renderer_fill_rect(renderer, (double)divider_x, 6.0, 1.0, (double)(h - 12), 0xFF404045);

// Settings toggle icon
int settings_x = toggle_start;
struct BLImageCore settings_img = {0};
if (icon_load("preferences-system", dock_icon_size, &settings_img)) {
    blend_renderer_draw_image(renderer, &settings_img, (double)settings_x, (double)cy);
}

// Launcher toggle icon  
int launcher_x = settings_x + dock_icon_size + DOCK_PAD;
struct BLImageCore launcher_img = {0};
if (icon_load("system-search", dock_icon_size, &launcher_img)) {
    blend_renderer_draw_image(renderer, &launcher_img, (double)launcher_x, (double)cy);
}
```

Update `dock_icon_at()`:
```c
int toggle_start = start_x + total_w + DOCK_PAD;
int settings_x = toggle_start;
int launcher_x = settings_x + dock_icon_size + DOCK_PAD;

if (mouse_x >= launcher_x && mouse_x < launcher_x + dock_icon_size) return -3;
if (mouse_x >= settings_x && mouse_x < settings_x + dock_icon_size) return -2;
```

## main_shell.zig launcher panel (blend2d)

Constants (matching cairo-pango):
```zig
const LAUNCHER_W: i32 = 520;
const LAUNCHER_H: i32 = 420;

```

State variables:
```zig
var launcher_surface = SurfaceState{ .height = 0 };
var launcher_open = false;
var launcher_hover_idx: i32 = -1;
var launcher_scroll: i32 = 0;
var pointer_on_launcher = false;
```

Key functions to port:
- `toggleLauncher()` — create/hide launcher surface
- `launcherVisibleRows()` — calculate visible row count
- `launcherItemAt()` — hit-test grid cells
- `launchApp()` — execute via `c.system()`
- `renderLauncherPanel()` — draw using BlendRenderer

Surface creation in `main()`:
- Layer: `ZWLR_LAYER_SHELL_V1_LAYER_TOP`
- Anchor: BOTTOM | LEFT | RIGHT
- Keyboard interactivity: 1 (for Escape/Enter)
- Initially hidden (size 0,0)
